# Portfolio Intelligence

Progetto Python per l'analisi di portafogli di investimento basati su dati di mercato.
Rappresenta un portafoglio come insieme di posizioni (ticker + peso) e fornisce utilità di base per validarlo.
Usa pandas, numpy e yfinance per il recupero e l'elaborazione dei dati finanziari.

## Uso

```
python main.py -p AAPL:0.5 -p MSFT:0.5 --period 1y
```
